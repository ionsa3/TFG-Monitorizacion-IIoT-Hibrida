# Arquitectura Híbrida Segura para Monitorización IIoT
**Trabajo de Fin de Grado - Grado en Ingeniería Informática (UNIR)**

Este repositorio contiene la implementación técnica de una arquitectura híbrida para la monitorización industrial utilizando:
- **Infraestructura como Código:** Terraform.
- **Contenedores:** Docker.
- **Simulación:** Python + MQTT.
- **Seguridad:** VPN (IPsec) y mTLS.

## Estado del proyecto
Actualmente en fase de **Diseño y Configuración de Infraestructura** (Entrega borrador 1 - 08/04/2026).
